## older_work
- 어르신 일자리 매칭 웹

## 소개
서비스 개요
공공 일자리 API를 활용하여 어르신(만 65세 이상)에게 맞는 일자리를 추천하는 웹 서비스입니다.

배경 / 개발 이유
고령화 사회가 진행됨에 따라 많은 어르신들이 활동적인 사회 참여 기회와 경제적 자립을 필요로 합니다. 이에 어르신들이 쉽게 적합한 일자리를 찾을 수 있도록 지원하고자 서비스를 개발했습니다.

해결하고자 하는 문제
- 정보 접근성이 낮아 일자리 찾기가 어려운 어르신들의 어려움
- 공공 일자리 정보가 흩어져 있어 효율적 매칭이 어려운 문제
- 어르신들의 사회적 참여 부족과 활동 기회 제한 문제
